CUP

CUP 어플리케이션을 안드로이드,IOS용으로 개발했습니다.

CUP은 PDF 문서의 특정 영역을 잘라내서 스크랩후 재편집할 수 있고, 네트워크상의 다른 사용자와 공유할 수 있는 기능을 제공합니다.
